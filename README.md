# saphnet-pterodactyl-images
Custom Docker images for Pterodactyl, for use in the Sapphic Homelab/Home Server

Most of the files here come from [pyrodactyl-oss/images](https://github.com/pyrodactyl-oss/images). For the Java images, the difference in the Dockerfiles here and there are that this repository adds the `webp` package, for Dynmap to be able to use webp.
